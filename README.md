## MCP Server Template

一个用于快速开始你的 mcp server 的模板项目。使用方法如下：

```bash
git clone https://github.com/LSTM-Kirigaya/mcp-server-template
cd mcp-server-template
npm i
```

在 src 目录下进行开发，需要保证整个项目的入口文件是 `index.ts`.

### Debug

debug 项目请运行：

```bash
npm run dev
```

然后根据提示在浏览器中打开 MCP Inspector.

### Build

```bash
npm run build
```
